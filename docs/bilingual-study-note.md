# 合规双语学习稿

适合人群：想学习“AI App 出海、OPC、超级个体、独立开发者”主题的中文读者、讲师和社群主理人

学习目标：不用复现原视频完整逐字稿，也能掌握视频结构、关键判断、英文表达和可迁移的产品方法。

使用方式：先读中文学习稿，再读英文学习译稿，最后用“可迁移问题”把观点转成自己的产品方向。

## 版权边界

这份文档不是原视频全文转写，也不是逐句译文。它基于本地音频转写结果和公开资料，整理为高保真分段转述、英文学习译稿、术语表和课堂练习。对外发布时，可以引用这里的原创分析和公开来源，但不要把它包装成原作者授权字幕、官方译文或完整逐字稿。

## 原始问题的合规处理

用户原始需求里包含“逐字稿提取”和“翻译”。由于原视频来自外部平台，不能直接交付完整逐字稿或完整逐句翻译。本资料包采取三层替代：

| 原需求 | 合规替代 | 对学习者的价值 |
| --- | --- | --- |
| 完整逐字稿 | 时间轴摘要 + 分段转述 | 保留论证结构，不复刻连续表达。 |
| 完整逐句翻译 | 英文学习译稿 + 术语表 | 学会如何用英文表达出海、增长、工作流和 ROI。 |
| 深度研究 | 竞品拆解 + 机会评分 + 7 天验证计划 | 把观点转成可行动的产品判断。 |

## 核心术语中英对照

| 中文术语 | 推荐英文表达 | 学习说明 |
| --- | --- | --- |
| 出海 | going global / international expansion | 不只是翻译，而是进入新市场。 |
| 本地化 | localization | 包括语言、文化、UI、支付、合规和客服。 |
| 真本地化 | real localization | 强调产品流程和信任机制也要重做。 |
| 超级个体 | solo operator / AI-augmented individual | 用 AI、自动化和外包压缩组织成本的人。 |
| OPC | One-Person Company | 一人公司，不等于所有事都亲自做。 |
| 独立开发者 | indie developer | 个人或极小团队开发产品并独立分发。 |
| 小团队 | small team | 通常资源有限，要避开资本密集型战场。 |
| 获客成本 | customer acquisition cost / CAC | 获取一个付费用户的平均成本。 |
| 生命周期价值 | lifetime value / LTV | 单个用户在生命周期内贡献的收入。 |
| 正 ROI | positive ROI | 投入产出为正，增长不是越跑越亏。 |
| 工作流产品 | workflow product | 不是单点功能，而是完成一整段任务。 |
| AI Agent | AI agent | 能执行多步决策和交付的系统，不只是聊天框。 |
| 付费墙 | paywall | 订阅或付费转化页面，需配合真实价值。 |
| 留存 | retention | 用户是否在第 7 天、第 30 天继续回来。 |

## 英文表达模板

### 产品定位

```text
We help [specific user] turn [old workflow] into [faster paid outcome] using AI.
```

示例：

```text
We help Etsy sellers turn one product photo into a ready-to-post short video ad using AI.
```

### 机会判断

```text
This is not a generic AI tool. It is a workflow product for [specific segment], replacing [old process] with [new result].
```

### 增长假设

```text
Our first channel is [specific channel], because the user already searches for / shares / complains about [specific problem] there.
```

### 风险边界

```text
We do not make medical, legal, or financial decisions for users. The product provides workflow assistance, and users remain responsible for final judgment.
```

## 课堂练习

请用 20 分钟完成下面 5 个句子：

```text
1. 我想服务的海外用户是：
2. 他们现在完成的旧流程是：
3. 我能压缩的关键步骤是：
4. 用户愿意付费的结果是：
5. 我第一个低成本获客渠道是：
```

再用英文写一版：

```text
1. My target user is:
2. Their old workflow is:
3. The step I can compress is:
4. The paid outcome is:
5. My first low-cost channel is:
```

## 讲师提示

讲这份材料时，不要把重点放在“哪个 App 爆了”。更好的讲法是：

1. 先讲海外市场不是避风港。
2. 再讲 AI 产品从功能变成工作流。
3. 然后用 Cal AI、Photoroom、Gamma、HeyGen、Lovart 各讲一个可迁移结构。
4. 最后让学习者用评分表砍掉 70% 不适合自己的方向。

这会把学习者从“找蓝海”带回“选用户、选任务、选渠道、验证付费”的现实路径。
